WMU

WMU 是一套基于 ratchet 开发的UI库。旨在统一和规范移动端开发，避免重复造轮子，提升开发效率而打造的移动端 UI 框架。

轻量的UI库

WMU 是非常轻量的，提供了20个左右常用的组件。

对于只有HTML&CSS的组件，你只需要复制HTML代码既可以使用。他的大部分JS组件都是独立的 Zepto 插件，并且提供了 Zepto/jQuery 风格的API，你将会非常熟悉这种方式。
